# PhiloMind Frontend

PhiloMind is a professional personal philosophical assistant leveraging AI. This repository contains the Next.js web application deployed to Hugging Face Spaces.

## Tech Stack
- **Framework**: Next.js 14
- **Styling**: TailwindCSS & Vanilla CSS
- **Integration**: Supabase & OpenRouter AI
